# 问题笔记.md

# 安装新版本的vue-cli 3.0
```bash
npm install -g @vue/cli   # 开始安装

vue -V / vue --version  ## 查看版本

npm uninstall -g @vue/cli  ## 写在安装

```

# 创建工程
```bash
vue create project  ## 创建项目

npm run serve

```

# 使用stylus
```bash
npm install -g stylus stylus-loader # 下载安装

npm install ## 重新建立依赖

# package.json中配置后， 项目中就可以用了。
"dependencies": {
    "stylus": "^0.54.5",
    "stylus-loader": "^3.0.2",
    ........
  },

```
